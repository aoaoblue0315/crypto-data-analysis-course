# crypto-data-analysis-course

Pythonで学ぶ暗号資産データ分析入門 — データ取得・統計分析・シグナル構築・バックテストまで全10巻で体系的に解説

## 概要

暗号資産のトレードに興味があり、感覚や経験則ではなくデータに基づいた判断を身につけたい方のための実践講座です。

Binance APIからのデータ取得に始まり、統計分析、指標の構築、バックテスト、統計的検証まで、データドリブンなトレード戦略の構築プロセスを一通り体験できます。すべてのコードはJupyter Notebook上で動かしながら学べます。

## 対象読者

- 暗号資産のトレードに興味があり、データに基づいた判断を身につけたい方
- Pythonの基礎文法（変数・ループ・関数）が分かる程度の方
- 暗号資産や金融の事前知識は不要です

## カリキュラム

| # | タイトル | テーマ |
|---|---------|--------|
| Vol.01 | データ取得とクレンジング | 分析の第一歩。信頼できるデータを手に入れる |
| Vol.02 | リターン分布と出来高分布 | 価格変動と出来高の統計的な性質を理解する |
| Vol.03 | 相関分析 | 複数銘柄間の関係性を定量的に把握する |
| Vol.04 | リードラグ分析 | 銘柄間の「先行・遅行」関係を定量的に検証する |
| Vol.05 | テクニカル指標の作成と評価 | テクニカル指標を自分で実装し、統計的な性質を評価する |
| Vol.06 | 指標の探索 | 価格データの外に目を向け、「使える指標」を見つけ出す |
| Vol.07 | シグナルから損益曲線へ | 指標の特性を深く検証し、体系的にバックテストで評価する |
| Vol.08 | 時系列分割による頑健性検証 | バックテスト結果を「信じてよいのか？」を検証する |
| Vol.09 | 5分位分析による指標評価 | 指標がリターンを予測できるかを5分位分析で評価する |
| Vol.10 | ブートストラップによる統計的検証 | 戦略の成績が「たまたま」なのか「統計的に有意」なのかを検証する |

## 前半（Vol.01〜05）と後半（Vol.06〜10）の構成

**前半: データの理解と基礎スキル**

Vol.01〜05では、データの取得・クレンジングから統計分析、テクニカル指標の構築まで、データ分析の基礎を固めます。ここで身につけたスキルが、後半の戦略構築の土台になります。

**後半: 戦略構築と検証**

Vol.06〜10では、前半で学んだ手法を応用して、実際にトレーディングシグナルを構築し、バックテストと統計的検証を行います。「本当に使えるのか？」を厳密に問い続ける姿勢を学びます。

## 必要な環境

- Python 3.10 以上
- Jupyter Notebook または JupyterLab

各notebookの冒頭で必要なライブラリを `pip install` しているため、事前のインストール作業は不要です。

### 使用ライブラリ

| ライブラリ | 用途 |
|-----------|------|
| requests | Binance REST APIの呼び出し |
| pandas | データ操作 |
| numpy | 数値計算 |
| matplotlib | グラフ描画 |
| japanize-matplotlib | matplotlibの日本語対応 |
| scipy | 統計検定 |

## 使い方

# リポジトリをクローン

cd crypto-data-analysis-course

Vol.01から順番に進めることを推奨します。各巻は前の巻の内容を前提として構成されています。

## 注意事項

- 各notebookはBinance公開APIからリアルタイムにデータを取得します。APIキーは不要ですが、インターネット接続が必要です。
- 分析対象の銘柄や期間はnotebook内で自由に変更できます。
